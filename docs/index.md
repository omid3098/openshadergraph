# OpenShaderGraph Docs

OpenShaderGraph is a TypeScript-first shader graph with a Bun server, ReactFlow UI, and compiler-driven language packs.

## Key Concepts

- Canonical data in `data/nodes/**` and `data/languages/**`
- ThreeJS GLSL is always compiled for preview
- Graph (de)serialization is UI-agnostic and stable

## Getting Started

```bash
bun install
bun run dev
```

Docs are available at `/docs` when built.
