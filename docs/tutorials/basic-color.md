# Basic Color

In this tutorial, you’ll connect a constant color to the fragment output.

## Steps

1. Add a `Color` node (`constants/color`).
2. Add `FragmentOutput`.
3. Connect `Color.out` → `FragmentOutput.Albedo`.
4. Tweak the color and see the preview update.

That’s it — you now have a solid base to extend with textures or math nodes.

Example graph: `examples/basic_color.json`.
