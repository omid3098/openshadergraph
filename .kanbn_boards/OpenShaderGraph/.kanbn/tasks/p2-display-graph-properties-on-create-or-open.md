# P2 - Display graph properties on create or open

When we create a new graph, the properties panel will not show the graph properties. we have to add a new node, select it and deselect it for be able to see the graph properties in the properties panel.
The same thing happens when closing a graph or opening a new graph. we have to select/deselect a node in the graph to see the graph properties.
